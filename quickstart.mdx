## Installation

To install the xCommand CLI, run the following command in your terminal:

```bash
curl -s https://xcommand.ai/install.sh | bash
```

Verify the installation:

```bash
xcommand version
```
## Basic Usage 

Run the CLI in interactive mode:

```bash
xcommand
```
